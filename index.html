Lesson 5: Variables and Strings

Variables

variables can be used to store values using an assignment statement (Name = Exoression) that must start with a letter or an underscore but can contain numbers.
String variables are assigned by surrounding a list of characters in single or double quotes It is a convention to start variable names with lowercase letters.
Variables can store the result of calculations and can depend on the value of other variables or even the value of itself (eg. days = days -1)

Comments

Comments can be written by preceeding text with a #

Decimals

If a decimal point is used in a calculation or definition of a variable used in a calculation then the exact answer is given else the integer answer is given.

Strings, operations and indexes

Whereas '+' is used to add numbers together, it is also used to join strings together. Strings can also be multiplied (e.g. exclamation = '!'*5 will create a variable containing the string !!!!!).
Strings have an index or position. The index of the string starts from 0 in the first position of the string and increases by 1 as we move to the characters further right.
A part of a string can ne extracted or referenced using an index(e.g. name = "Joe" s=name[0] print s will print J).
Indexs can be used to select parts of strings (e.g. print name[0:1] will print Jo). Leaving the start (or end) position blank means the first (or last) position of the string is selected.
Prefixing the position with a '-' selects the postion of the string that many positions from the end (eg print name[:-1] gives Jo)

Find

The find operator can be used to find the position of the first occurence of a substring. -1 is output if the substring is not found.
A second parameter can be added to the find operator. This parameter tells the find operator which position to start searching for the substring from.
Find is an example of a built-in procedure.

Lesson 6: Functions

What is a function?

A function does some operations on one or more inputs to produce one or more outputs.

What is the difference between making and using a function?

Making a function involves writing the operatoins that will take place om the inputs to the function. Using a function involves calling the function on the inputs to produce the output

How do functions help programmers avoid repetition?

Functions are a stored procedure that a programmer can call to run pre-written code anywhere in a program without having to type the code out again.

What happens if a function doesn't have a return statement?

If a function doesn't have a return stateent then nothing is outputted. The function has no purpose and doesn't make a difference in the code.

Loops
